### **📊 Ежедневные выплаты**
```typescript
// Автоматическое создание ежедневных выплат
const dailyPayout = await prisma.dailyPayout.create({
  data: {
    date: today,
    totalAmount: 0,
    totalUsers: 0,
    status: 'PENDING'
  }
});

// Обработка всех инвестиций, заканчивающихся сегодня
const expiringInvestments = await prisma.investment.findMany({
  where: {
    status: 'ACTIVE',
    endDate: {
      gte: today,
      lt: tomorrow
    }
  }
});
```

### **👥 Мониторинг активности**
```typescript
// Система расчета активности
const ACTIVITY_WEIGHTS = {
  DEPOSIT: 50,           // Депозит - высший приоритет
  SLOT_PURCHASE: 30,     // Покупка слота
  LOTTERY_TICKET: 10,    // Покупка лотерейного билета
  REFERRAL_ACTIVITY: 20,  // Активность рефералов
  DAILY_LOGIN: 5,        // Ежедневный вход
  WITHDRAWAL: -10,       // Вывод средств (снижает активность)
  INACTIVITY: -5         // Неактивность
};
```

### **❄️ Автоматическая заморозка**
```typescript
// Критерии для заморозки:
// 1. Не делал депозит
// 2. Его рефералы тоже неактивны  
// 3. Низкий activity score
// 4. 10+ дней неактивности
const usersToFreeze = inactiveUsers.filter(user => {
  const hasInactiveReferrals = user.referrals.every(ref => !ref.hasMadeDeposit);
  return !user.hasMadeDeposit && hasInactiveReferrals && user.activityScore < 10;
});
```

## 🚀 **API ЭНДПОИНТЫ**

### **Ежедневные выплаты**
- `GET /api/admin/today-payouts` - Сегодняшние выплаты
- `POST /api/admin/process-today-payouts` - Обработать выплаты
- `GET /api/admin/daily-payouts` - История выплат

### **Управление пользователями**
- `GET /api/admin/active-users` - Активные пользователи
- `GET /api/admin/inactive-users` - Неактивные пользователи
- `POST /api/admin/freeze-accounts` - Заморозить аккаунты
- `POST /api/admin/auto-freeze-inactive` - Автозаморозка

### **Мониторинг активности**
- `GET /api/admin/activity-stats` - Статистика активности
- `POST /api/admin/recalculate-activity` - Пересчет активности
- `GET /api/admin/dashboard-stats` - Общая статистика
