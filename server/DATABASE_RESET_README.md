# 🗑️ Database Reset Scripts

Этот проект включает скрипты для полного сброса базы данных, что полезно для тестирования или очистки всех данных пользователей.

## 🚀 Быстрый запуск

### Через NPM скрипт (рекомендуется)
```bash
# В корневой директории проекта
pnpm run db:reset --filter server
```

### Через shell скрипт (Linux/Mac)
```bash
# В директории server/
./reset-db.sh
```

### Через PowerShell скрипт (Windows)
```powershell
# В директории server/
.\reset-db.ps1
```

## 📋 Что делает скрипт сброса

### 🗑️ Удаляемые данные:
- ✅ Все пользователи (`users`)
- ✅ Все кошельки (`wallets`)
- ✅ Все слоты майнинга (`mining_slots`)
- ✅ Все транзакции (`withdrawals`, `investments`, `swap_transactions`)
- ✅ Все реферальные данные (`referral_earnings`)
- ✅ Все уведомления (`notifications`)
- ✅ Все логи активности (`activity_logs`)
- ✅ Все лотерейные билеты (`lottery_tickets`)
- ✅ Все завершенные задачи (`completed_tasks`)
- ✅ Все аккаунты заморозки (`account_freezes`)
- ✅ Курсы обмена (`exchange_rates`)
- ✅ Задачи (`tasks`)

### 👑 Создаваемые данные:
После сброса автоматически создается:
- ✅ Админ-пользователь с балансом 50,000 USD
- ✅ Реферальный код для админа
- ✅ Дефолтный курс обмена (1.0)
- ✅ Один тестовый слот майнинга для админа

## ⚠️ Важные предупреждения

1. **Бэкап данных**: Перед запуском скрипта убедитесь, что у вас есть резервная копия важных данных
2. **Остановка сервера**: Убедитесь, что сервер остановлен перед запуском сброса
3. **Потеря данных**: Все пользовательские данные будут безвозвратно удалены

## 🔧 Технические детали

### Структура скрипта:
- `src/scripts/resetDatabase.ts` - основной TypeScript скрипт
- `reset-db.sh` - shell скрипт для Linux/Mac
- `reset-db.ps1` - PowerShell скрипт для Windows

### Зависимости:

### Порядок удаления:
Скрипт удаляет данные в правильном порядке, учитывая внешние ключи:
1. AccountFreeze
2. Investment
3. Withdrawal
4. ReferralEarning
5. Notification
6. Wallet
7. MiningSlot
8. CompletedTask
9. ActivityLog
10. LotteryTicket
11. SwapTransaction
12. User (последний, так как связан со всеми остальными)

## 🚨 Troubleshooting

### Ошибка "Server is running":
```
⚠️  Warning: Server process detected. Please stop the server before resetting database.
```
**Решение:** Остановите сервер перед запуском сброса:
```bash
# Если есть скрипт остановки
pnpm run stop

# Или вручную найдите и остановите процесс
```

### Ошибка подключения к базе данных:
```
❌ Error resetting database: Connection error
```
**Решение:** Убедитесь, что:
- База данных запущена
- Переменные окружения DATABASE_URL корректны
- У вас есть доступ к базе данных

### Ошибка прав доступа:
```
❌ Error: Permission denied
```
**Решение:** Убедитесь, что у вас есть права на выполнение скриптов и доступ к базе данных.
