## 🔧 **BACKEND API**

### **DELETE /api/admin/delete-user/:userId**
```typescript
// Полное удаление пользователя и всех связанных данных
await prisma.$transaction(async (tx) => {
  // 1. Удаляем логи активности
  await tx.activityLog.deleteMany({ where: { userId } });
  
  // 2. Удаляем уведомления
  await tx.notification.deleteMany({ where: { userId } });
  
  // 3. Удаляем кошельки
  await tx.wallet.deleteMany({ where: { userId } });
  
  // 4. Удаляем слоты
  await tx.miningSlot.deleteMany({ where: { userId } });
  
  // 5. Удаляем инвестиции
  await tx.investment.deleteMany({ where: { userId } });
  
  // 6. Удаляем выводы
  await tx.withdrawal.deleteMany({ where: { userId } });
  
  // 7. Удаляем реферальные доходы
  await tx.referralEarning.deleteMany({ where: { userId } });
  
  // 8. Удаляем заморозки аккаунта
  await tx.accountFreeze.deleteMany({ where: { userId } });
  
  // 9. Удаляем лотерейные билеты
  await tx.lotteryTicket.deleteMany({ where: { userId } });
  
  // 10. Удаляем выполненные задачи
  await tx.completedTask.deleteMany({ where: { userId } });
  
  // 11. Удаляем транзакции обмена
  await tx.swapTransaction.deleteMany({ where: { userId } });
  
  // 12. Обновляем рефералов (убираем referredById)
  await tx.user.updateMany({
    where: { referredById: userId },
    data: { referredById: null }
  });
  
  // 13. Наконец, удаляем самого пользователя
  await tx.user.delete({ where: { id: userId } });
});
```

## 🚀 **ИСПОЛЬЗОВАНИЕ**

### **1. В админ панели:**
1. Перейти на вкладку "Delete Users" 🗑️
2. Найти пользователя через поиск
3. Нажать "Delete User"
4. Указать причину удаления
5. Подтвердить удаление

### **2. API запрос:**
```bash
DELETE /api/admin/delete-user/{userId}
Content-Type: application/json

{
  "adminId": "ADMIN_PANEL",
  "reason": "Violation of terms"
}
```

### **3. Ответ API:**
```json
{
  "success": true,
  "message": "User @username has been completely deleted from the database",
  "data": {

    "totalInvested": 100.0,
    "totalEarnings": 30.0,
    "deletedAt": "2024-01-01T00:00:00.000Z"
```
