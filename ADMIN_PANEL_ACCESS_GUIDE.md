## 🚀 **КАК ИСПОЛЬЗОВАТЬ**

### **1. Вход в админ панель:**
1. Откройте приложение
2. На главной странице найдите карточку "Admin Panel" 🛠️
3. Нажмите на неё
4. Введите пароль: `admin123`
5. Нажмите "Login to Admin Panel"

### **2. Навигация:**
- **Overview** - общая статистика
- **Daily Payouts** - управление выплатами
- **User Management** - управление пользователями
- **Activity Monitor** - мониторинг активности
- **Delete Users** - удаление пользователей

### **3. Удаление пользователя:**
1. Перейдите на вкладку "Delete Users" 🗑️
2. Найдите пользователя через поиск
3. Нажмите "Delete User"
4. Укажите причину удаления
5. Подтвердите удаление

## ⚙️ **НАСТРОЙКА**

### **Изменение пароля админа:**
```bash
# В .env файле
ADMIN_PASSWORD=your_new_password
```

### **Настройка JWT секрета:**
```bash
# В .env файле
JWT_SECRET=your_jwt_secret_key
```

## 🔧 **ТЕХНИЧЕСКИЕ ДЕТАЛИ**

### **Backend API:**
- `POST /api/admin/login` - аутентификация
- `GET /api/admin/verify-token` - проверка токена
- `DELETE /api/admin/delete-user/:userId` - удаление пользователя
- `GET /api/admin/dashboard-stats` - статистика
- `GET /api/admin/daily-payouts` - ежедневные выплаты
- `GET /api/admin/active-users` - активные пользователи
- `GET /api/admin/inactive-users` - неактивные пользователи
- `POST /api/admin/freeze-accounts` - заморозка аккаунтов

### **Frontend компоненты:**
- `AdminPanel.tsx` - главная страница админ панели
- `AdminDashboard.tsx` - дашборд с вкладками
- `DailyPayoutsPanel.tsx` - управление выплатами
- `UserActivityMonitor.tsx` - мониторинг активности
- `UserDeletionPanel.tsx` - удаление пользователей
