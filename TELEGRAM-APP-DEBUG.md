# 🔧 Telegram App Debug Guide

## Проблема
React Error #310 не позволяет приложению открываться в Telegram.

## Решение

### 1. Тестирование упрощенной версии

**Шаг 1: Создайте простую сборку**
```bash
cd client
pnpm run build:simple
```

**Шаг 2: Проверьте локально**
```bash
pnpm run preview
```

**Шаг 3: Если работает локально, деплойте на Render**
Используйте команду сборки: `pnpm run build:simple`

### 2. Альтернативные команды сборки

**Без минификации (рекомендуется):**
```bash
pnpm run build:debug
```

**Обычная production сборка:**
```bash
pnpm run build:prod
```

**Обычная сборка:**
```bash
pnpm run build
```

### 3. Проверка в Telegram

1. **Откройте приложение в Telegram**
2. **Проверьте консоль браузера** (если доступна в Telegram WebView)
3. **Убедитесь, что нет ошибок загрузки**

### 4. Отладка

**Если приложение все еще не открывается:**

1. **Проверьте URL приложения в Telegram боте**
2. **Убедитесь, что сервер запущен и доступен**
3. **Проверьте CORS настройки**
4. **Проверьте, что все статические файлы загружаются**

### 5. Минимальная конфигурация для Telegram

Для максимальной совместимости с Telegram WebView используйте:

```bash
pnpm run build:simple
```

Эта команда:
- Отключает минификацию
- Использует простые компоненты
- Убирает сложную логику аутентификации
- Создает максимально совместимую сборку

### 6. Проверка файлов

После сборки убедитесь, что в `dist/` есть:
- `index.html`
- `assets/` папка с JS и CSS файлами
- Все файлы загружаются без ошибок

### 7. Тестирование в разных браузерах

Перед деплоем протестируйте в:
- Chrome (основной браузер для WebView)
- Firefox
- Safari (если поддерживается)

### 8. Логирование

Добавьте в консоль браузера:
```javascript
console.log('App loaded successfully');
```

### 9. Если ничего не помогает

Попробуйте:
1. **Полностью отключить минификацию**
2. **Использовать более старую версию React**
3. **Проверить совместимость с Telegram WebView API**

## Команды для быстрого тестирования

```bash
# Создать простую сборку
cd client && pnpm run build:simple

# Проверить локально
pnpm run preview

# Очистить и пересобрать
pnpm run clean && pnpm run build:simple
```
