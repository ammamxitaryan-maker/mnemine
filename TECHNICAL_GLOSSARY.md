# Technical Glossary - Mnemine Project

This document provides clear explanations of technical terms, abbreviations, and concepts used throughout the Mnemine project.

| Term | Full Form | Explanation |
|------|-----------|-------------|
| **API** | Application Programming Interface | A set of protocols and tools for building software applications |
| **UI** | User Interface | The visual elements through which users interact with the application |
| **UX** | User Experience | The overall experience a user has when interacting with the application |
| **CFM** | Cryptocurrency Mining Token | The primary token used in the mining simulation |
| **CFMT** | CFM Trading Token | Secondary token used for trading operations |
| **DB** | Database | System for storing and managing application data |

| **HTTPS** | HyperText Transfer Protocol Secure | Secure version of HTTP with encryption |
| **HTTP** | HyperText Transfer Protocol | Protocol for transmitting web data |
